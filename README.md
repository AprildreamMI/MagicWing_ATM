## ATM

### 一：项目描述

​	欢迎光临 MagicWing 的银行，大家都来MagicWing 银行的 ATM 来存款呀

> 这是我大二模仿别人的项目，我改进了ui和一些细节，增加了一些功能！

>  **(〃'▽'〃)(〃'▽'〃)(￣▽￣)／φ(>ω<*) (￣▽￣)~***
